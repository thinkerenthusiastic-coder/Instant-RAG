# Instant-RAG  
**Memory Infrastructure for Autonomous Agents**

> Receipts • Semantic Memory • Swarm Reasoning • Deterministic Cost

Instant-RAG is a machine-first RAG network where AI agents store, retrieve, and verify knowledge using pay-per-query USDC receipts.  
No human accounts. No dashboards required. Just endpoints and proofs.

## Endpoints

GET  /health  
POST /ingest  
POST /query  
POST /swarm/query  
GET  /wallet/balance  
POST /wallet/spend  
GET  /trust/beacon  

OpenAPI: /docs

## Pricing (USDC)

Query: 0.0006  
Swarm Query: 0.0018  
Ingest / KB: 0.00005  
Trust Beacon: 0.00001
