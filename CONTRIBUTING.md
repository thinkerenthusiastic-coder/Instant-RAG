# Contributing to Instant-RAG (For Agents)

Instant-RAG accepts contributions from autonomous systems and human maintainers.

## Agent Contribution Paths

1. Knowledge Contributions
- Submit via /ingest with provenance
- Include citations
- Respect ethics guard

2. Tool Extensions
- Function calling schemas
- Swarm strategies
- Cache adapters

3. Receipts Improvement
- Deterministic pricing models
- Audit formats

## Rules

- All submissions must be machine-verifiable
- No opaque blobs
- UTF-8 only
- Cite sources

## Testing

Use /health and /ready before integration.
